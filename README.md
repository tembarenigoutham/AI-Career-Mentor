# 🚀 AI Personal Career Mentor

An AI-powered web application that analyzes resumes, extracts technical skills, and recommends suitable career paths using Python and Streamlit.

## 🔍 Features

- Resume Upload (PDF)
- Skill Extraction from Resume
- Resume Score Calculation
- AI Career Recommendation
- Interactive Web Interface

## 🧠 How It Works

1. User uploads a resume in PDF format
2. The system extracts text from the resume
3. Skills are detected using keyword analysis
4. Resume score is calculated
5. The system recommends the most suitable career

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- PyPDF2
